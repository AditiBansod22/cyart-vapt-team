About the Project

This repository contains my work for the Vulnerability Assessment and Penetration Testing (VAPT) assignment.
The aim of this project is to understand basic security concepts, perform vulnerability scanning using open-source tools, and document the results properly.

The work is divided into two sections:

Theoretical Study
Practical Implementation
Theoretical Part

The theory section focuses on understanding the core concepts of VAPT.

Topics Covered
Security Assessment and its types
VAPT methodology (Planning, Scanning, Exploitation, Reporting)
Security standards like GDPR, HIPAA, ISO 27001
Risk assessment using CVSS scoring
Common vulnerabilities (SQL Injection, XSS, open ports)
Basics of documentation and reporting

This part helped in building a strong foundation before performing practical tasks.

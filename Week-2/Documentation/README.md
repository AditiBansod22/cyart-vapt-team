# Vulnerability Assessment Documentation

This folder contains the main documentation of the vulnerability scanning activity performed during the VAPT task.

The report explains the process followed during the assessment, including target identification, network scanning, web vulnerability scanning, and analysis of the findings.

The documentation includes:

* Introduction to vulnerability assessment
* Tools used during the scanning process
* Results of the Nmap network scan
* Results of the Nikto web vulnerability scan
* Basic remediation recommendations

This report provides a structured overview of the security issues identified during the scanning process.

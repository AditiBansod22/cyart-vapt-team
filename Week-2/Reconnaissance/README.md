# Reconnaissance Documentation

This folder contains the reconnaissance report created during the initial phase of the vulnerability assessment process.

Reconnaissance is the process of gathering information about the target system before performing deeper security analysis. In this task, the IP address of the Metasploitable2 machine was identified and basic network information was collected.

Tools such as ifconfig, Nmap, and Nikto were used to gather information about the system's network configuration, open ports, and available services.

The reconnaissance report provides a summary of the information collected and explains how the scanning tools were used to identify potential attack surfaces on the target machine.

# Nikto Web Server Scan

This folder contains the results of the web vulnerability scan performed using the Nikto tool.

Nikto is a web server scanner that identifies potential vulnerabilities, outdated software, and security misconfigurations in web services.

Command used:

nikto -h http://<target_ip>

The scan was conducted on the HTTP service running on the Metasploitable2 machine. The results highlighted possible security issues such as outdated server versions and missing security configurations.

The screenshot included in this folder represents the Nikto scan output captured during the vulnerability scanning process.

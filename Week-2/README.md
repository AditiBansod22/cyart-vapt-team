# Week 2 – VAPT Practical Task

## Overview

This folder contains the practical work and documentation completed during Week 2 of the CYART VAPT training program. The objective of this task was to perform reconnaissance and vulnerability scanning on a vulnerable virtual machine using Kali Linux security tools.

## Target Environment

Target Machine: Metasploitable2
Testing Platform: Kali Linux

## Tools Used

* Nmap (Network scanning)
* Nikto (Web server vulnerability scanning)
* Kali Linux
* Metasploitable2

## Tasks Performed

1. Identification of target IP address using ifconfig.
2. Network scanning using Nmap to identify open ports and services.
3. Web vulnerability scanning using Nikto.
4. Documentation of reconnaissance activities and scan results.

## Repository Structure

Nmap_Scan – Contains Nmap scan results
Nikto_Scan – Contains Nikto scan results
Reconnaissance – Contains reconnaissance documentation
Documentation – Contains vulnerability scanning report
Screenshots – Supporting screenshots

##  Theoretical Work

The theoretical section includes detailed explanations of the following topics:

- Advanced Exploitation Techniques  
- API Security Testing  
- Privilege Escalation and Persistence  
- Network Protocol Attacks  
- Mobile Application Penetration Testing  
- Reporting and Remediation  

Each topic is explained with:
- Definition  
- Key Concepts  
- Example  
- Tools  
- Conclusion  



##  Practical Work

The practical section includes hands-on labs performed using various tools and vulnerable environments.

###  Modules Covered:
1. Advanced Exploitation  
2. API Security Testing  
3. Privilege Escalation  
4. Network Attacks  
5. Mobile Application Testing  

###  Tools Used:

- Burp Suite  
- DVWA  
- LinPEAS

##  Capstone Project

A complete penetration testing simulation was performed on a vulnerable machine.

### Activities Performed:
- Network scanning  
- Vulnerability identification  
- Exploitation  
- Privilege escalation  
- Reporting  

###  Deliverables:
- Final Report (Detailed findings and remediation)  
- Non-Technical Summary (Simple explanation for stakeholders)  
- Screenshots (Proof of work)

- Developed reporting and documentation skills

#  Practical Work – Week 4 (VAPT)

##  Overview
This section contains the practical implementation of various Vulnerability Assessment and Penetration Testing (VAPT) concepts.  
Each lab focuses on identifying, exploiting, and documenting different types of security vulnerabilities in a controlled environment.

---

##  Modules Covered

The practical work is divided into the following five modules:

1. **Advanced Exploitation**  
2. **API Security Testing**  
3. **Privilege Escalation**  
4. **Network Protocol Attacks**  
5. **Mobile Application Testing**  

Each module has its own folder containing reports, checklists, and screenshots.

---

## 🛠 Tools and Technologies Used

The following tools were used during the practical sessions:

- Kali Linux  
- Metasploit Framework  
- Nmap  
- Burp Suite  
- DVWA (Damn Vulnerable Web Application)  
- LinPEAS  
- Responder & Ettercap  
- Wireshark  
- MobSF  
- Frida  

---

##  Methodology Followed

The following steps were followed in each practical task:

## 1. Setup
- Configured virtual lab environment  
- Connected attacker (Kali Linux) and target machines  

## 2. Reconnaissance
- Performed scanning using Nmap  
- Identified open ports and running services  

## 3. Vulnerability Identification
- Analyzed services and applications  
- Identified possible vulnerabilities  

## 4. Exploitation
- Used tools like Metasploit and Burp Suite  
- Executed attacks to gain access or extract data  

## 5. Post-Exploitation
- Performed privilege escalation  
- Captured sensitive information  

## 6. Documentation
- Recorded all steps performed  
- Captured screenshots as proof  
- Prepared reports with findings and remediation  



## Contents of Each Module

Each module folder includes:

- **Report.pdf**  
  Contains objective, tools used, steps performed, output, and result  

  

- **Screenshots**  
  Contains proof of execution such as commands, tool outputs, and successful exploitation
